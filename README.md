# RecycleTrack

A web application for tracking recycling activities.

## Tech Stack

**Frontend:**
- React
- Vite
- Tailwind CSS v4

**Backend:**
- Django
- Django REST Framework

## Setup

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
source env/Scripts/activate  # On Windows Git Bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
